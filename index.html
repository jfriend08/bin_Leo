A suite of tools to address common questions raised in genomic studies — mostly with regard to overlap and proximity relationships between data sets — BEDOPS aims to be scalable and flexible, facilitating the efficient and accurate analysis and management of large-scale genomic data.

Current download: v2.2.0 / May 22, 2013 (changes)

A companion set of documentation and sample input files is also available for offline browsing.





  • bedops - apply set operations on any number of BED inputs
  • bedextract - efficiently extract BED features
  • closest-features - matches nearest features between BED files
  • bedmap - map overlapping BED elements onto target regions, and optionally compute any number of common statistical operations
  • sort-bed - apply lexicographical sort to BED data
  • starch and unstarch - compress and extract BED data
  • starchcat - merge compressed archives
  • conversion tools - convert common genomic formats to BED

Please browse our usage examples page, which shows the BEDOPS suite in use for answering day-to-day research questions.

Also, please join our dedicated user forum, where we field your usage and other support questions!
